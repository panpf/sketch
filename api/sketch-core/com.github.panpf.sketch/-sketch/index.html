Sketch

class Sketch

A service class that performs an ImageRequest to load an image.

Sketch is responsible for handling data acquisition, image decoding, image conversion, processing cache, request management, memory management and other functions.

You just need to create an instance of the ImageRequest subclass and pass it to the enqueue or execute method for execution.

Sketch is designed to be sharable and works best when the same instance is used throughout the application via the built-in extension function Context.sketch
