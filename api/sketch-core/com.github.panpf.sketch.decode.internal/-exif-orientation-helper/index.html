val rotationDegrees: Int

Returns the rotation degrees for the current image orientation. If the image is flipped, i.e., .isFlipped returns true, the rotation degrees will be base on the assumption that the image is first flipped horizontally (along Y-axis), and then do the rotation. For example, .TRANSPOSE will be interpreted as flipped horizontally first, and then rotate 270 degrees clockwise.
