open class DrawableDecoder(requestContext: RequestContext, dataSource: DrawableDataSource, mimeType: String?) : Decoder

Extract the icon of the installed app and convert it to Bitmap

The following decoding related properties are supported:

  • sizeMultiplier

  • precisionDecider

  • scaleDecider

  • colorType
