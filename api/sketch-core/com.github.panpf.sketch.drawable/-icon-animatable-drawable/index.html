IconAnimatableDrawable

class IconAnimatableDrawable(val icon: Drawable, val background: Drawable? = null, val iconSize: <Error class: unknown class>? = null, @ColorInt val iconTint: Int? = null) : IconDrawable, Drawable.Callback, Animatable2Compat, SketchDrawable

It consists of two parts: icon and bg. bg is scaled to fill bounds, the icon size is unchanged always centered. It is suitable for use as a placeholder image for waterfall flow.
