rememberAsyncImagePainter

@Composable
fun rememberAsyncImagePainter(uri: String?, sketch: Sketch, state: AsyncImageState = rememberAsyncImageState(), alignment: Alignment = Alignment.Center, contentScale: ContentScale = ContentScale.Fit, filterQuality: FilterQuality = DefaultFilterQuality): AsyncImagePainter

Return an AsyncImagePainter that executes an ImageRequest asynchronously and renders the result.

**This is a lower-level API than AsyncImage and may not work as expected in all situations. **

  • AsyncImagePainter will not finish loading if AsyncImagePainter.onDraw is not called. This can occur if a composable has an unbounded (i.e. Constraints.Infinity) width/height constraint. For example, to use AsyncImagePainter with LazyRow or LazyColumn, you must set a bounded width or height respectively using Modifier.width or Modifier.height.

  • AsyncImageState.painterState will not transition to PainterState.Success synchronously during the composition phase. Use SubcomposeAsyncImage or set a custom ImageRequest.Builder.size value (e.g. size(Size(100, 100))) if you need this.

Parameters

uri

ImageRequest.uri value.

sketch

The Sketch that will be used to execute the request.

state

AsyncImageState that will be used to store the state of the request.

alignment

Optional alignment parameter used to place the AsyncImagePainter in the given bounds defined by the width and height.

contentScale

Used to determine the aspect ratio scaling to be used if the canvas bounds are a different size from the intrinsic size of the image loaded by uri. This should be set to the same value that's passed to Image.

filterQuality

Sampling algorithm applied to a bitmap when it is scaled and drawn into the destination.
