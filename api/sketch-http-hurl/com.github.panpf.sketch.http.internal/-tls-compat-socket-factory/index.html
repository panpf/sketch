TlsCompatSocketFactory

class TlsCompatSocketFactory(val enabledTlsProtocols: Array<String>)

Enables TLS v1.1, v1.2, v1.3 when creating SSLSockets.

Android supports TLS v1.1, v1.2 from API 16, but enables it by default only from API 20. Android supports TLS v1.3 from API 16, but enables it by default only from API 29.
