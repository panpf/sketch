ScaledAnimatableDrawable

class ScaledAnimatableDrawable @JvmOverloads constructor(val drawable: Drawable, val fitScale: Boolean = true) : Drawable, Animatable2Compat, Drawable.Callback, SketchDrawable

A Drawable that centers and scales its drawable to fill its bounds.

This allows drawables that only draw within their intrinsic dimensions (e.g. AnimatedImageDrawable) to fill their entire bounds.
